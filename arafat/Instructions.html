Leaky Memory

Suppose you have two FNImageView objects that need to access on another. 

@interface FNImageView : UIImageView

@property (nonatomic, strong) FNImageView *imageView;

@end

You find that your memory usage keeps increasing each time you open the ViewController that loads the two FNImageView objects.

@interface ViewController ()

@property (nonatomic) FNImageView *aImageView;
@property (nonatomic) FNImageView *anotherImageView;

@end

@implementation ViewController

- (void)viewDidLoad {
    [super viewDidLoad];
    // Do any additional setup after loading the view.
    aImageView                      = [FNImageView new];
    anotherImageView                = [FNImageView new];

    aImageView.imageView            = anotherImageView;
    anotherImageView.imageView      = aImageView;
}

- (void)dealloc
{
    self.aImageView         = nil;
    self.anotherImageView   = nil;
}

Fix the memory problem but keep all of the functionality

Slow Download

Suppose you are downloading a image to display as an avatar image for a user in a messaging app. The code below works but causes the UI to freeze during the download, image conversion, and image resizing.

// Download the image
NSURL *url = [NSURL URLWithString:@"http://lorempixel.com/400/200/"];
NSURLRequest *req = [NSURLRequest requestWithURL:url];
NSURLResponse *res = [[NSURLResponse alloc] init];
NSError *err = nil;
NSData *data = nil;
data = [NSURLConnection sendSynchronousRequest:req
                             returningResponse:&res
                                         error:&err];

// Convert the data to a UIImage
UIImage *image = [UIImage imageWithData:data];

// Scale the image
UIImage *thumbImage = nil;
CGSize newSize = CGSizeMake(90, (90 / image.size.width) * image.size.height);
UIGraphicsBeginImageContext(newSize);
[image drawInRect:CGRectMake(0, 0, newSize.width, newSize.height)];
thumbImage = UIGraphicsGetImageFromCurrentImageContext();
UIGraphicsEndImageContext();

// Add this to a UIImageView
self.imageView.image = thumbImage;

Prevent this code from interrupting the UI.

Delegation

You are given FNTextFieldViewController that contains a UITextField and is presented modally over FNMasterViewController.

Objective 1

Create: A delegate protocol on FNTextFieldViewController and set FNMasterViewController as it's delegate.

Objective 2

Create a method that sends the value of the text displayed in the UITextField to the delegate.

Given: A user types "h" then "i" so that the UITextField reads "hi"

Return: Delegate receives "h" and then "hi"

Given: A user presses backspace when the textField reads "hi"

Return: Delegate receives "h"

Objective 3

Given: A user types "dismiss" in the text field

Reponse: Delegate hides the keyboard and dismisses the FNTextFieldViewController.

Blocks

Suppose some users have been using fake words as their last name. You have been tasked with finding any usage of the words: "awesome", "crap" and "dinosaur". Do your work in the NSString+BlockQuestion category.

Objective 1

You need to create a method that returns a BOOL on if an array of last names contains one (or more) of the fake words.

Given: 

@[@"appleseed", @"Worku", @"Crap", @"Nguyen", @"Smith"]

Return: TRUE

Objective 2

Add a block to the method that has the NSIndexSet of the fake names. Note this should only be called when fake names are found.

Given: 

@[@"Appleseed", @"Worku", @"Crap", @"Nguyen", @"Smith", @"awesome"]

Return: TRUE and executes the block with an index set containing indexes 2 and 5.

Localization

A server is supplying data, to an iOS app, about a deal from an orange producer. The data has been mapped into an object called FNOrangeDeal. Do your work in the NSString+LocalizationQuestion category.

Your task is to create a method that will take the data and return a human readable string.

Objective 1

Display the price localized to the currency that the deal is in.

Given [FNOrangeDeal bangladeshiOranges]:

[[FNOrangeDeal alloc] initWithAmount:5 currency:@"bdt" forPrice:30000];

Return: The output string should be: "The oranges cost ৳300.00 for 5."

Objective 2

The output string should be localized.

Given [FNOrangeDeal americanOranges]:

[[FNOrangeDeal alloc] initWithAmount:10 currency:@"usd" forPrice:400];

Return:

On a German language iOS device: "Die Orangen kosten $4.00 für 10."

On a English language iOS device: "The oranges cost $4.00 for 10."
